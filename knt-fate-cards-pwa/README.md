# KNT Fate Cards — PWA

แอปสุ่มเลขที่นักเรียนและเปิดไพ่ชะตา สำหรับครูนิวตรอน โดยเว็บไซต์สาธารณะไม่มีข้อมูลส่วนบุคคลของนักเรียน

## ความสามารถ

- นำเข้าไฟล์รายชื่อส่วนตัวจากอุปกรณ์ แล้วเลือกห้องก่อนเริ่มสุ่ม
- แสดงเลขที่พร้อมชื่อ–นามสกุลหลังนำเข้า
- ระบบไม่สุ่มซ้ำ เลขที่ขาดเรียน และประวัติแยกรายห้อง
- ไพ่ชะตาใบใหญ่ พร้อมเสียงลุ้นและเสียงเปิดไพ่
- ติดตั้งบนหน้าจอโฮม iPhone, iPad และ Mac
- ทำงานออฟไลน์หลังเปิดใช้งานครั้งแรก
- รายชื่อและการตั้งค่าถูกเก็บเฉพาะในอุปกรณ์ด้วย Local Storage

## ความเป็นส่วนตัว

Repository นี้ไม่มีรายชื่อนักเรียน ไฟล์รายชื่อส่วนตัวต้องได้รับจากครูนิวตรอนโดยตรงและนำเข้าผ่านหน้าแอป ข้อมูลจะไม่ถูกส่งกลับมายัง GitHub หรือเซิร์ฟเวอร์อื่น

## ติดตั้งบน iPhone หรือ iPad

1. เปิดลิงก์แอปด้วย Safari
2. แตะปุ่มแชร์
3. เลือก **เพิ่มไปยังหน้าจอโฮม**
4. แตะ **เพิ่ม**

## ติดตั้งบน Mac

เปิดลิงก์แอปด้วย Safari แล้วเลือก **File > Add to Dock** หรือเปิดด้วย Chrome แล้วเลือกติดตั้งแอปจากแถบที่อยู่

> PWA ต้องเปิดผ่าน `https://` หรือ `localhost` จึงจะติดตั้งและทำงานออฟไลน์ได้ การเปิด `index.html` โดยตรงเหมาะสำหรับทดสอบเท่านั้น
